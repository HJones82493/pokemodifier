# Pokemodifier

The purpose of this is to allow easy ROM modification of the Pokemon games. For copyright purposes (go Nintendo!) original roms will not be supplied.
The plan is to eventually extend this app to a web based app and to support later generations, but this is an iterative process by a single developer at the moment, so your patience is appreciated. 
**bear in mind that this app is for modifying the ROM itself, not the in game RAM or save file values**
